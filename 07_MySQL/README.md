```mysql
SHOW DATABASES;
```

: 현재 서버에 어떤 DB들이 있는지 보여준다.

```mysql
USE database_name;
```

: 사용할 DB를 지정한다.

```mysql
SHOW TABLEs;
```

: DB의 테이블 이름을 보여준다.

```mysql
SHOW TABLE STATUS;
```

: DB의 테이블 정보를 보여준다.

```mysql
DESC city;
```

: city 테이블에 무슨 열이 있는지 확인한다.
